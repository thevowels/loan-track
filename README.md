# Loan Track

### Project Description

The Purpose of the porejct is to create a system help the user to track their loan records.

The user can Add Peoples and track the loan/return records for each person.



### Database Structure


##### Table Structures

- **User**
    - id

- **People**
    - id
    - name
    - phone
    - picture
    - address
    - user_id

- **Transactions**
    - id
    - user_id
    - person_id
    - amount
    - datetime
